Created for hands-on learning with pytest

**Program:** SENAI

This repository contains a collection of exercises, programming challenges, and test practices focused on pytest, all written in Python. The goal is to strengthen logical reasoning, programming skills, and knowledge of automated testing.

### Installing Dependencies

Before running the code and tests, install the dependencies listed in requirements.txt:

```pip install -r requirements.txt```
### Running Tests with pytest

To run the automated tests, use pytest in the terminal within the project directory:

```pytest```

You can use any IDE or code editor, such as Visual Studio Code or PyCharm, to open and modify the Python files.
